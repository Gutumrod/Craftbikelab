## 📋 Prompt สำหรับ Hermes (แปลง xlsx → CSV)

```
ฉันมีไฟล์ Excel ที่ export มาจาก Bigsaler (Shopee format) 
ชื่อไฟล์: test.xlsx
อยู่ที่: C:\Users\Win10\Downloads\เอกสาร\test.xlsx

โครงสร้างไฟล์ต้นทาง:
- 1 สินค้า = หลายแถว (แต่ละแถวคือ 1 variant เช่น size/สี)
- คอลัมน์สำคัญ: ชื่อสินค้า, รายละเอียดสินค้าแบบเต็ม (HTML), 
  รายละเอียดสินค้าแบบย่อ, ลิงก์แหล่งสินค้า, ราคา, ส่วนลด,
  รูปภาพสินค้า 1-9 (URL จาก Shopee CDN)

ให้ทำตามขั้นตอนนี้:

1. MERGE VARIANTS
   - Group แถวที่มีชื่อสินค้าเดียวกันเข้าด้วยกัน → เหลือ 1 แถวต่อสินค้า
   - ราคา: เก็บราคาต่ำสุด (หลังหักส่วนลด)
   - รูป: เก็บ URL รูปแรกที่มีค่า (รูปภาพสินค้า 1)
   - ลิงก์: เก็บ URL เดิม 1 URL เป็น original_link

2. CLEAN
   - ลบ HTML tag ออกจากรายละเอียดสินค้าแบบเต็ม ให้เหลือแค่ text
   - ย่อชื่อสินค้าให้กระชับ ตัดคำซ้ำและ SEO keyword ที่ไม่จำเป็นออก
     ตัวอย่าง: "ท้ายรถมอเตอร์ไซค์ กระเป๋าท้ายรถ กันน้ำ 40L/66L ใช้เป็นกระเป๋าเป้สะพายได้สิ่งสำคัญสำหรับการตั้งแคมป์"
     → "กระเป๋าท้ายรถ กันน้ำ 40L/66L"

3. CLASSIFY
   category: เลือก 1 ค่าจากนี้เท่านั้น
   - crashbar (แคชบาร์, กันล้ม)
   - exhaust (ท่อไอเสีย, slip-on)
   - light (ไฟ, LED, DRL)
   - handle (แฮนด์, กริ๊ป)
   - bag (กระเป๋า)
   - screen (วินด์สกรีน, บังลม)
   - rear_rack (แร็คท้าย)
   - side_rack (แร็คข้าง)
   - sticker (สติ๊กเกอร์, decal)
   - top_box (กล่องท้าย)
   - tire (ยาง)
   - accessory (อุปกรณ์เสริม)
   - other (ไม่เข้าหมวดไหน)

   is_universal:
   - true = ถ้าชื่อสินค้าไม่ได้ระบุรุ่นรถเฉพาะเจาะจง
   - false = ถ้าชื่อสินค้ามีชื่อรุ่นรถ เช่น PCX, Aerox, Forza, Click, Wave, Fino, NMAX, Zoomer, CBR, CRF

   model_slugs:
   - ถ้า is_universal = true → ว่าง
   - ถ้า is_universal = false → ระบุรุ่นรถที่เจอในชื่อสินค้า เช่น ["pcx160", "aerox155"]

4. OUTPUT เป็นไฟล์ CSV ชื่อ output.csv มี columns ดังนี้:
name, brand, price, image_url, category, is_universal, model_slugs, 
shopee_link, original_link, deeplink_status, spec

กฎ:
- shopee_link = เว้นว่าง (จะทำ deeplink ทีหลัง)
- deeplink_status = "pending"
- brand = ดึงจากชื่อสินค้า ถ้าหาไม่ได้ให้ว่าง
- spec = รายละเอียดสินค้าแบบย่อ (text ที่ clean แล้ว ไม่เกิน 300 ตัวอักษร)
- model_slugs = ถ้ามีหลายรุ่น ให้คั่นด้วย | เช่น pcx160|aerox155

บันทึกไฟล์ output ที่: C:\Users\Win10\Downloads\เอกสาร\output.csv
```
