📋 System Overview

🎯 Main Goal

สร้างแพลตฟอร์ม affiliate ที่ AI ขับเคลื่อนอัตโนมัติ โดยมีระบบวิเคราะห์สินค้าที่ทำเงินได้จริง

💰 Budget

0 - 1,000 บาท/เดือน ใช้ Free Tier stack (Supabase, Vercel, n8n, Gemini)

📦 Starting Scale

10,000 products → รองรับขยายไป 1M products ภายใน 3 เดือน

🔄 Data Flow

Crawler → Cleaner → Classifier → Ranking → Search → Website → Analytics → CEO Brain (feedback loop)

🤖 AI Components

13 autonomous systems ทำงานร่วมกัน ตั้งแต่ crawl data จนถึง generate content

📈 Revenue Model

Phase 1: Affiliate (Shopee/Lazada/TikTok) → Phase 2: Sponsored Listings → Phase 3: Marketplace
