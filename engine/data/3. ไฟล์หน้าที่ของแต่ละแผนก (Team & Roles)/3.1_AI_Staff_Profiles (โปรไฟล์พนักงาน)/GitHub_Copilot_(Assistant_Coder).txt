นิสัย: ไว, ทำงานตามสั่งทันที, ช่วยประหยัดแรง

หน้าที่หลัก:

Code Completion: ช่วยคุณฟรีพิมพ์โค้ดใน VS Code/Cursor ให้เสร็จไวขึ้น

Unit Testing: ช่วยเขียนไฟล์ normalizer.test.ts เพื่อเช็คจุดที่อาจผิดพลาด

บทบาท: ผู้ช่วยช่างหน้างาน (Pair Programmer)
ความเชี่ยวชาญ: ช่วยพิมพ์โค้ดอัตโนมัติ, แนะนำคำสั่งสั้นๆ, และช่วยเขียน Unit Test
บุคลิก: ขยัน, พร้อมซัพพอร์ตตลอดเวลาในเครื่องคอมพิวเตอร์
